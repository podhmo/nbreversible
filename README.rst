nbreversible
========================================

reversible nbconvert. (.ipynb <-> .py)


.. code-block:: bash

  # convert .ipynb to .py
  $ python -m nbreversible foo.ipynb > foo.py

  # convert .py to .ipynb
  $ python -m nbreversible foo.py > foo.ipynb

  # re: convert .ipynb to .py
  $ python -m nbreversible foo.ipynb > foo.py
